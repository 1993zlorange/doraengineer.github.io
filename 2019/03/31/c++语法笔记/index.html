c++语法笔记

2019-03-31 学习笔记

第一次学习c++是在大二，也是编程学的第一门语言，学习的时候大多时间花在理解概念，所以说第一门语言基础打好了学习其他语言都是大同小异，相同的概念再看下语法就可以了。c++属于基础语言，因为专业是系统工程，大多课程需要应用算法进行系统优化，运筹等，比较注重算法效率，因此课程安排了c++语言，至今也觉得比较受用，因为很多硬件设备提供的都是c++接口，因此c++在做算法以及硬件实现方面都十分实用。都说c++难，可能难在基础语言，以及指针等一些概念上，这些都是只有做算法才会用上，一般反正如果自己做小应用的时候是很少用上这些概念，最多的就是控制，循环，函数，面向对象等。现在很多语言也是基于c++语言演化来的例如python，手感更好但是毕竟演化经过了包装，效率肯定不及c++，就循环而言就慢很多。

  • 多态

    c++使用指针和虚函数实现多态，因为默认的赋值运算符并不会操作虚函数表，虚函数也就是基类能直接指向派生类对象的函数，大家名字都一样不过不同的对象实现不一样

一接口多实现（避免多”if”繁琐语句），使派生类同时完成一个动作

animal *p[10];
for(int i=0;i<100;i++){
    p[i]->shout();
    p[i]->run();
}

多态性polymophism->继承

Dog d,*pd;
Animal a,*pa;//pa可指Dog类，因为Animal是Dog基类
pa=pd;
pa->move();//Animal move

地址调用函数为a->b(),普通对象调用函数为a.b();

一个多态实现的例子

void recognize(Animal * pa){
    pa->shout();
    pa->run;
}//recognize函数根据不同的动物指针pa来实现跑和叫，当然跑和叫函数需要在不同动物类型下具体定义
Animal *pselect[10];//实例化10个动物
for(int i=0;i<10;i++){
    recognize(pselect[i]);
}//10个动物依次跑和叫
